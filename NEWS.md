# DRMLSurv (development version)

## 0.1.0

### New features
- Added core functions for censoring-score estimation, matching-based imputation, and policy learning.
- Added `ComputeScores()`, `get_doublescores()`, and `impute_censored_outcomes()` workflows.
- Added `rfdtr()` and `Drmatch()` for dynamic treatment regime learning.

### Documentation
- Added roxygen2 documentation.
- Added README and package badges.

### Infrastructure
- Added test suite with `testthat`.
- Added GitHub Actions for R CMD check and coverage.
